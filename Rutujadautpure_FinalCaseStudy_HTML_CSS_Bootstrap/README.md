# Project Title
Responsive Customer Engagement Portal


# Project Overview
This project is developed as part of the Final Case Study using HTML5, CSS3, Bootstrap 5, and JavaScript. 

The goal of this project is to design and implement a fully responsive and accessible customer engagement portal that works smoothly across devices and modern browsers. The portal presents company information, displays services, collects user input through validated forms, and includes interactive behavior using JavaScript.


# Features Implemented

# HTML (Structure & Semantics)
- Proper HTML5 document structure
- Semantic elements (header, section, nav, footer)
- Organized content using block and inline elements
- Accessible form labels and media elements
- Embedded audio and video with controls

# CSS (Styling & Layout)
- External stylesheet for maintainability
- Custom styling using selectors and box model
- Responsive layout using Flexbox
- Clean and consistent UI design

# Bootstrap Components
- Responsive navigation bar
- Cards for content display
- Styled table for services
- Pagination component
- Form validation styling
- Responsive grid system

# JavaScript Functionality
- DOM manipulation
- Event handling
- Form validation
- Dynamic content updates
- Fetch API for asynchronous data handling
- Proper error handling


# Responsiveness
The portal is fully responsive and tested on:
- Desktop
- Tablet
- Mobile devices

Bootstrap grid system and flexible layouts are used to ensure adaptability.


# Accessibility (A11Y)
- Proper label usage for form inputs
- Alt text for media (where applicable)
- Readable font sizes and contrast
- Semantic structure for better screen reader support


# Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Fetch API


# How to Run the Project
1. Extract the ZIP folder.
2. Open the project folder.
3. Open `index.html` in Google Chrome or Microsoft Edge.
